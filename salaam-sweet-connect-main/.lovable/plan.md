
## النظام الذكي لإدارة معامل الهندسة 🔬

### نظرة عامة
نظام ويب متكامل ومستقل يعمل مع Flask + MySQL خارجي، يخدم نوعين من المستخدمين: **المشرف** بصلاحيات كاملة، و**الطالب** بصلاحيات تصفح وطلب.

---

### 🔐 نظام المصادقة (Authentication)
- **صفحة تسجيل الدخول الموحدة**: نموذج بسيط (اسم المستخدم + كلمة السر)
- **دور المشرف (Admin)**: وصول كامل للوحة التحكم
- **دور الطالب (Student)**: وصول لواجهة التصفح والطلبات فقط
- تحديد الدور تلقائياً بعد تسجيل الدخول وتوجيه كل مستخدم للوحته المناسبة

---

### 🎓 واجهة الطالب (Student Portal)

**الصفحة الرئيسية - كتالوج القطع:**
- شريط بحث ذكي (بالاسم أو الكود)
- فلترة حسب الفئة (حساسات، ميكروكنترولر، أسلاك، إلخ)
- بطاقات تعرض: اسم القطعة، الصورة، الكمية المتاحة، موقعها (رقم الدرج)
- مؤشر بصري واضح: **متاح** (أخضر) / **غير متاح** (أحمر)

**صفحة القطعة:**
- تفاصيل القطعة كاملة
- زر "طلب استعارة" (يُرسل طلباً للمشرف)

**صفحة طلباتي:**
- قائمة بعهد الطالب الحالية والسابقة
- حالة كل عهدة: نشطة / مرجعة / متأخرة
- تاريخ الاستعارة وتاريخ الإرجاع المتوقع

---

### 🛠️ لوحة تحكم المشرف (Admin Dashboard)

**الصفحة الرئيسية - الإحصائيات:**
- إجمالي القطع في المخزن
- عدد العهد النشطة
- عدد العهد المتأخرة (مع تنبيه بصري)
- آخر النشاطات (سحب/إرجاع)

**إدارة المخزون:**
- جدول شامل لكل القطع مع إمكانية البحث والفلترة
- إضافة قطعة جديدة (اسم، كود، الكمية، رقم الدرج، الفئة)
- تعديل الكميات وبيانات القطع
- عرض موقع كل قطعة (الدرج)

**إدارة العهد (Loans Management):**
- تسجيل عهدة جديدة: إدخال الرقم الجامعي → يظهر اسم الطالب → اختيار القطعة → تحديد تاريخ الإرجاع
- التحقق التلقائي من أن الطالب ليس لديه عهد متأخرة
- زر "إرجاع" بضغطة واحدة مع تحديث المخزون تلقائياً
- فلترة العهد: نشطة / متأخرة / مكتملة

**سحب المعسكرات (Camps Checkout):**
- إنشاء طلب معسكر باسم الجهة/الفعالية
- إضافة قائمة قطع متعددة دفعة واحدة بكميات محددة
- إدارة استرجاع القطع بعد انتهاء المعسكر

**إدارة الطلاب:**
- عرض قائمة الطلاب
- إضافة طالب جديد بالرقم الجامعي والاسم
- حظر طالب أو رفع الحظر عنه

---

### 📄 التوثيق والتقارير
- **إيصال PDF للعهدة**: إقرار استلام يحتوي اسم الطالب، الرقم الجامعي، القطعة، التاريخ، وخانة التوقيع
- **تقرير العهد النشطة**: PDF شامل بكل العهد غير المُرجعة مع أسماء الطلاب وتواريخ الإرجاع

---

### 🎨 تصميم الواجهة
- **لغة عربية كاملة** مع دعم RTL
- ألوان هادئة ومهنية (أزرق داكن + رمادي + لون تمييزي)
- تصميم متجاوب يعمل على الموبايل والحاسوب
- واجهة سهلة الاستخدام بدون تعقيد

---

### 🔌 التكامل مع Backend
- الواجهة الأمامية تتصل بـ Flask API الخاص بك
- جميع البيانات تُجلب من MySQL عبر API endpoints
- متغير بيئي لعنوان الـ API يسهل التعديل لاحقاً
- في البداية ستعمل ببيانات وهمية (Mock Data) حتى يجهز الـ Flask API
